📥 Importar usuarios desde archivo

Sube un archivo Excel (.xlsx) o CSV con la lista de usuarios. Las columnas deben ser: username, name, email, role, password (opcional). Roles válidos: admin, technician, employee.

⚠️ Los usuarios se crean para tu empresa actual. Si password está vacío, se usa demo. Si el username ya existe en esta empresa, se omite (no se sobrescribe).
